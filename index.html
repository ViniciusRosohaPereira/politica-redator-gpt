Política de Privacidade – Redator Crítico GPT

Última atualização: 21 de abril de 2025

Este GPT, desenvolvido por Vinícius Rosoha Pereira, utiliza webhooks para acessar informações públicas de forma automatizada e gerar respostas com base em temas políticos, econômicos, sociais, religiosos e tecnológicos.

Coleta de Dados

Este GPT não coleta, armazena ou processa informações pessoais do usuário. As informações utilizadas são extraídas exclusivamente de fontes públicas fornecidas pelo usuário ou pelo sistema n8n via webhooks.

Uso das Informações

As respostas geradas são baseadas em dados estruturados recebidos via APIs de terceiros, como fluxos de notícias do tipo RSS/XML convertidos por automações do usuário. Nenhuma informação pessoal é manipulada.

Armazenamento de Dados

Não há banco de dados vinculado, cookies ou qualquer rastreamento embutido neste GPT. As interações são descartadas automaticamente após o término da sessão.

Segurança

Todos os fluxos de dados passam por conexões seguras (HTTPS) e não são compartilhados com terceiros. O usuário tem total controle sobre os endpoints utilizados.
