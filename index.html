Política de Privacidade – Redator Crítico GPT

Atualizado em: 21 de abril de 2025

1. Coleta de dados

Este GPT utiliza webhooks externos fornecidos pelo usuário (n8n) para obter informações públicas de notícias. Nenhuma informação pessoal é solicitada, processada ou armazenada.

2. Fontes de terceiros

As respostas geradas pelo GPT podem incluir conteúdo originado de feeds públicos RSS processados via automação externa (n8n.cloud), hospedada por n8n.io.

3. Segurança

A comunicação entre este GPT e os servidores externos utiliza HTTPS. Contudo, o usuário é responsável por verificar a segurança de qualquer conteúdo proveniente de links externos.

4. Integrações

Este GPT se integra com endpoints via webhooks configurados manualmente pelo autor (Vinícius Rosoha), para a coleta e envio de dados em tempo real.

5. Consentimento

Ao utilizar este GPT, você concorda com esta política. O uso das ações integradas requer autorização explícita no momento da execução, por medida de segurança do ChatGPT.
